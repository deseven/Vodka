Basic Usage

First of all, we need to define some basic site parameters: 

$params["system"] = array(
	"show_errors" => true,
	"clean_unused_vars" => true
);
Pages: 
// you should create the necessary files and place it in the
// correct locations, relative to the site root

$params["pages"] = array(
	array(
		"path" => "pages/main.html",
		"title" => "main page"
	),
	array(
		"path" => "pages/sec.html",
		"title" => "second page"
	)
);
Templates: 
// the template should contain at least one file
// called template.html

// it can contain the following variables:
// {VODKA:HEAD} - where to append head tags
// {VODKA:MENU} - where to place the generated menu
// {VODKA:CONTENT} - loaded page content
// {VODKA:TITLE} - page title
// {VODKA:TEMPLATE} - template path (if needed)

$params["templates"] = array(
	"demo" => "tpl/demo"
);
And a basic menu: 
// selected_class will be appended to the
// menu item of the current page

$params["menus"] = array(
	"html" => '<li><a href="{VODKA:URL}" 
	id="{VODKA:NAME}" 
	class="{VODKA:CLASS}">{VODKA:TITLE}</a></li>',
	"selected_class" => "selected"
);
The next step is to include the class file and to initialize vodka: 
require "vodka.class.php";
$vodka = new vodka($params);
After that we will load the template and the page, build the menu and then output the generated page: 
$vodka->loadTemplate("demo");
$vodka->loadCurrentPage();
$vodka->buildCurrentMenu();
$vodka->buildCurrentPage();
That's it!

Check the demo sources to get some advanced usage examples.

Have fun with vodka! :)
